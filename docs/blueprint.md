# **App Name**: RiceCooker Pro Max 5TB

## Core Features:

- Web Interface: Serve a webpage with controls and status information for the rice cooker.
- Rice Dispensing: Control the servo to dispense rice for a configurable duration.
- Water Pumping: Activate the water pump relay for washing rice for a configurable duration.
- Cooking: Engage the heater relay to cook the rice for a configurable duration.
- Configuration: Allow setting the duration for pumping, dispensing, and cooking via the web interface.
- Status Display: Display the current status of the rice cooker (READY, PUMPING, DISPENSING, COOKING) and the time remaining for each stage on the web interface.
- Login Page: Require users to log in before accessing the rice cooker controls. Display device ID on login.

## Style Guidelines:

- Primary color: Electric Green (#E3E3E3) for a techy, retro feel, reminiscent of old computer terminals.
- Background color: Very dark desaturated green (#BF3131), almost black, to provide contrast with the text.
- Accent color: Cyan (#F9F5F0) to highlight important information and interactive elements.
- Body and headline font: 'PT Sans', a humanist sans-serif font suitable for both headlines and body text
- Use simple, geometric icons to represent the different states and actions, consistent with the tech theme.
- Maintain a minimalist layout with clear, centered elements to ensure readability and ease of use on a small screen.
- Incorporate subtle text animations and transitions to indicate state changes and provide visual feedback.
